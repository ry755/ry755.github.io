Welcome to fox32os!
This is fox32os running in a WASM build of the fox32 emulator.

NOTE: Any changes you make *will not be saved!!*
To save files to persistent storage, store them on disk 1, then
download disk 1 as an image file using the buttons below the
emulator screen.
Disk 1 will need to be formatted before you use it for the
first time. Open a terminal window using the shortcut on the
desktop and run this command:

    format 1 userdata

"userdata" can be replaced with any label you want to use for
disk 1.

A link to the fox32 Discord server can be found on the GitHub
organization readme.

Have fun!!
